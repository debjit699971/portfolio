## 🚀 Deployment
This project is hosted on **GitHub Pages**.  
To deploy your own:
1. Push your portfolio repo to GitHub.  
2. Go to `Settings → Pages`.  
3. Select `Branch: main → /(root)` and save.  
4. Your site goes live 🎉
